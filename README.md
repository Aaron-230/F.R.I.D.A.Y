# F.R.I.D.A.Y
**F.R.I.D.A.Y.** (or Financial Resource Intelligence &amp; Digital Asset Yielder) is
an AI Machine where you can ask it any question based on financial terms.

### 📍 Where to use it

- **Web:** Just click on the link on the repo and start asking your questions on financial needs

----------

### 🛠 Built With
    
-   **Language:** [Python 3.14.5](https://www.python.org/downloads/)

-   **Libraries:** Flask, Google GenAI, Markdown and OS

### 🤝 How to Contribute

We welcome contributions to refine the Sentinel's behavior and level design!

1.  **Fork** the project.
    
2.  **Create a Feature Branch** (`git checkout -b feature/ImprovedAI`).
    
3.  **Commit your Changes** (`git commit -m 'Made the AI Smart'`).
    
4.  **Push to the Branch** (`git push origin feature/ImprovedAI`).
    
5.  **Open a Pull Request**.
    

**Roadmap for Contributors:**

-   Implementing File Uploads for Assessing profit-loss approximations and many more.
    
-   Refining AI on financial approximations.
    
-   Adding Stock Market Data Functions for better data approximation.

----------

### 🚀 Deployment

To Deploy it for local testing:

1. Clone the Repository
2. Install all the Libraries
3. Place your API Key to the api_key=""
4. Run the code using python and click on the url to get to the Web UI
